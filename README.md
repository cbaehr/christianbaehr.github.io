# christianbaehr.com (GitHub Pages)

This repository hosts the source for `christianbaehr.com`, served via GitHub Pages.

## Quick setup
1. Repo name must be: `christianbaehr.github.io`
2. GitHub → **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
3. Custom domain: `christianbaehr.com`

## Editing
- Main content: `index.md`
- Profile image: `images/profile.jpg` (add this file)

## Notes
This site uses the **Minimal Mistakes** Jekyll theme via `remote_theme`.
